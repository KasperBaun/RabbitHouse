# Arbejdsplan — Kaninhus

Trin i byggerækkefølge, grupperet pr. byggesystem. Hus-zonen (X = 0..1500).
Løbegården er en separat struktur og står til sidst.

## 0. Forberedelse & opmåling ✅

- [x] Vælg + ryd placering, ret terræn af
- [x] Mål op, sæt galger og snor (2000 × 3000 mm)
- [x] Marker hjørner ved (0,0), (2000,0), (2000,3000), (0,3000)
- [x] Afsæt grøftebredde til udgravning

## 1. Fundament ✅

- [x] Udgrav rendegrøft ~30 cm bred × ~80 cm dyb
- [x] Stabilgrus 100 mm i bund, komprimer
- [x] Læg fundablok-ring, 4 skifter halvstensforbandt
- [x] Vandret armering bund + top, lodret i hjørner
- [x] Udstøb hulrum med beton, lad hærde
- [x] Sokkelpuds på blokke. Kontroller om hele fundamentet er vandret, og ret evt. af med mørtel inden pudsning.
- [x] Støb klaplag i fundamentets indre.
